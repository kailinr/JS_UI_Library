# Custom JS UI Library
This is a custom CSS and JavaScript UI library boilerplate project. The JS library includes modules for dropdown, tab switches, snackbar, and tooltips. The project is compiled with Webpack with Babel and CSS loaders.


## How to Use this Library
To use this boilerblate, simply clone this repo and download the npm dependencies inside the directory:

`npm install`

### Dev Dependency Specs
* "webpack": "^5.70.0" 
* "babel-loader": "^8.2.4",
* "css-loader": "^6.7.1",
* "style-loader": "^3.3.1",
